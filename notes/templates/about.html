About Notes Manager

Notes Manager is a simple and intuitive task management application built with Django and Bootstrap.

This application was created to help you organize your daily tasks efficiently. Whether you're managing personal tasks, work projects, or shopping lists, Notes Manager provides an easy-to-use interface to keep track of everything.

Key Features

  • Easy Task Creation: Add new tasks with just a few clicks
  • Inline Editing: Edit tasks directly without navigating to a new page
  • Status Tracking: Mark tasks as completed or incomplete with a single click
  • Visual Feedback: Color-coded tasks and toast notifications for all actions
  • Responsive Design: Works seamlessly on desktop, tablet, and mobile devices

Technologies Used

  • Django 5.2.9 - Backend Framework
  • Python - Programming Language
  • MySQL - Database
  • Bootstrap 5 - Frontend Framework
  • JavaScript - Client-side Scripting
  • Bootstrap Icons - Icon Library
